# Android Automation

This project provides Android device automation capabilities via ADB.
Designed for AI coding agents (Claude Code, Gemini CLI, Cursor, GitHub Copilot, etc.)

## When to use this skill

Use this when the user:
- Mentions Android, mobile, emulator, ADB, or simulator
- Wants to test, tap, scroll, or interact with mobile apps
- Needs element locators, UI hierarchy, or accessibility tree
- Wants to automate Android app testing
- Asks for Appium-like automation without Appium

## Capabilities

| Category | Examples |
|----------|----------|
| **Device** | List devices, boot/shutdown emulators, get device info |
| **Apps** | Install, uninstall, launch, stop apps |
| **Interaction** | Tap, scroll, swipe, find elements by text/ID/class |
| **Input** | Type text, press hardware buttons, open URLs |
| **Testing** | Screenshots, accessibility audits, visual diffs, log monitoring |
